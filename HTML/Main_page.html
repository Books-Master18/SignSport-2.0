Добро пожаловать на SignSport!

Подберём подходящий для вас вид спорта не по вашей физической форме, а по личностным особенностям, который определим используя для этого ваш почерк!

Программа SignSport 1.0 с помощью результатов графологического анализа почерка (т. е. картину вашего характера и личности) для определения вида спорта, который подходит по вашему характеру — а не только физическим данным. Это уникальный подход, основанный на научных связях между почерком, личностью и предрасположенностью к определённым видам спорта.

Этапы работы проекта:

  • Проанализировать почерк человека
  • Загрузить данные его характера в программу
  • Получить долгожданный результат!

Загрузите отчёт вашего характера по почерку — и получите рекомендацию по подходящему виду спорта прямо сейчас!
